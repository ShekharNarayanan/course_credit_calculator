# Course Credit Calculator

A simple web app for calculating student course credits.

## Run locally

Clone the repository:

```bash
git clone https://github.com/ShekharNarayanan/course_credit_calculator.git
cd course_credit_calculator
```

Start a local Python server:

```bash
python -m http.server 8000
```

Open the app in your browser:

```text
http://localhost:8000
```

## Notes

- No build step is required.
- The app runs from the static files in this repository.
- Use `Ctrl+C` in the terminal to stop the server.
